# Framework Settings Use Contextual Accordions

Framework settings use consistent contextual accordion sections for Colors, Semantic colors, Typography, Border radius, Spacing, and future preference systems. Each collapsed header pairs a canonical label with a compact current-value preview; expanding reveals only that system's controls. This keeps the narrow sidebar scannable while allowing the Framework to grow without bespoke section behavior.
